# Rights and Assets

## Hard rule

A sample may be used only if its owner and allowed use are known.

## Metadata required

- creator/owner
- source
- license
- commercial allowed: yes/no/unknown
- model training allowed: yes/no/unknown
- redistribution allowed: yes/no/unknown

## Default policy

Unknown rights = blocked for commercial use.

## Collaboration with sample owner

Before using a private library in DML, create a written agreement covering:

1. ownership
2. allowed use in product
3. training permissions
4. redistribution permissions
5. revenue split
6. exit terms
