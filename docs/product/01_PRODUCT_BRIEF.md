# Product Brief

## Problem

Current AI music tools often generate a finished result that feels impressive but closed. Professional musicians need control: tracks, stems, arrangement decisions, performance nuance, corrections, versions and export.

## Solution

DKey Music Layer turns human musical intent into a structured production project:

- song structure
- arrangement plan
- instrument map
- performance plan
- stems
- mix instructions
- project file

## Primary user

- musician/producer who thinks in performance and arrangement
- singer-songwriter who wants a real production workflow
- arranger keyboard user moving into AI-era production
- studio creator who wants mobile control over a full project

## First concrete product

A controllable music production foundation that can:

1. accept a musical intent prompt
2. create a DML project
3. create a track arrangement
4. connect tracks to sample packs
5. validate assets
6. render a smoke-level stem/mix plan
7. expose the project to DAW/mobile shells

## Product north star

The user says: “Make the chorus wider, keep the verse simple, give me real strings and export stems.”

The system understands, changes the project and produces a controllable result.
